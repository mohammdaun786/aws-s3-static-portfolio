# AWS S3 Static Portfolio

This project is a single-page static portfolio website built using HTML and CSS.

## Features
- Responsive single-page layout
- Hosted on AWS S3
- Served via CloudFront with HTTPS
- Clean and professional UI

## Tech Stack
- HTML5
- CSS3
- AWS S3
- AWS CloudFront
- Git & GitHub
